# F1 2025 Analytics

An end-to-end Formula 1 analytics project built using **FastF1 API, Python, and Power BI** to analyze the 2025 F1 season through race performance, lap pace, tyre strategies, qualifying metrics, and championship progression.

## Project Overview

This project was developed as an independent analytics portfolio project inspired by my passion for Formula 1. The objective was to build a complete data pipeline and dashboarding solution that transforms raw race data into meaningful performance insights for drivers, teams, and the championship as a whole.

Using race-by-race data from the **FastF1 API**, I collected and structured datasets covering lap times, driver performance, tyre usage, qualifying sessions, team standings, and championship metrics for the 2025 season. The data was then cleaned, transformed, modelled, and visualised in an interactive **Power BI dashboard**.

## Objectives

- Analyse driver and constructor performance across the 2025 season
- Compare lap pace, qualifying form, and tyre strategies
- Track championship progression over time
- Build a scalable data model for motorsport analytics
- Deliver an interactive dashboard for self-service insight exploration

## Tech Stack

- **Python** — data extraction, cleaning, preprocessing, and EDA
- **Pandas / PySpark** — data transformation and validation
- **FastF1 API** — race, session, and telemetry data
- **Power Query** — additional shaping and data preparation
- **Power BI** — data modelling, DAX measures, KPI reporting, dashboard development

## Data Sources

This project integrates multiple Formula 1-related datasets, including:

- FastF1 race and session data
- Lap time and performance metrics
- Tyre strategy data
- Driver and constructor metadata
- Championship standings
- Weather and contextual race information

## Project Workflow

1. Extracted 2025 Formula 1 race datasets from the FastF1 API
2. Cleaned and validated raw data in Python
3. Performed exploratory data analysis to identify trends and anomalies
4. Built relational data models linking race, driver, lap, team, and standings tables
5. Created DAX measures and KPIs in Power BI
6. Designed an interactive dashboard to explore season-wide insights

## Key Features

- 2025 season race analytics
- Driver performance comparison
- Constructor performance tracking
- Lap pace and consistency analysis
- Tyre strategy analysis
- Qualifying and race trend exploration
- Championship progression tracking
- Predictive modelling experiments for race winner forecasting

## Dashboard Pages

The dashboard includes four analytical views:

- **Championship Overview** — season progress, points trends, and title race development
- **Driver Performance** — pace, consistency, podiums, points, and comparative metrics
- **Team Performance** — constructor standings, points contribution, and performance comparison
- **Race & Tyre Strategy** — compound usage, stint behaviour, and race strategy insights
